# UK Fuel Prices Map

![Screenshot](screenshot.png)

A simple web app that displays a map of UK fuel prices using the latest data.

## Setup

```bash
# clone the repo

git clone https://github.com/markwylde/uk-fuel-price-map.git
cd uk-fuel-price-map

# install dependencies
npm i

# run the dev server
npm run dev
```

## Build

```bash
npm run build
```
